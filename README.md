# StackOverflowTag-Classcification
This is a text-Classification task to classify the tags of [azure, aws, gcp], based on the title of StackOverflow questions.  

Data is scraped from StackOverflow.  

## Dataset Details
* Azure: 77216 titles
* AWS: 82182 titles
* GCP: 17576 titles

## Requirements:
* Python 3.7
* TensorFlow 2.0
* Scikit-Learn
